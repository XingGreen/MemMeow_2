喵语转换规则

• 摩斯点（.） → 喵

• 摩斯划（-） → 呜

• 分隔符（/） → ……

作者想说

我是一名氛围编程工程师（Vibe Coding），这款软件是我和 Trae CN SOLO 一起协作开发的 GUI 项目，它还有几个可爱的名字：喵啊～、MemMeow、忆喵。

软件是用 HTML 开发的，已经用tauri打包成了 .apk 安卓安装包 和 .exe Windows 安装包，并且完全开源到 GitHub 社区，欢迎大家去提交 issues，提出你的改进建议。

这里也提醒一下：本软件仅用于娱乐，不负任何法律责任。

其实这款 MemMeow（摩斯喵语转译器），是我和 Trae CN SOLO 经历了几个小时的"击裂豆蒸"之后，终于开发出来的项目雏形。

为什么会做这个项目呢？

因为之前我把 AI 代写生成的摩斯喵语图片发到群里，有人说这样太浪费算力了，建议我直接用函数写一个。

再加上我自己也想用这种"摩斯喵语"来做趣味加密交流，于是就有了做这个软件的想法。

而且我很早之前就有一个心愿：做出一款完全属于自己的软件。

Vibe Coding 弥补了我技术力不足的问题，让我的想法能够快速实现。

在这里也特别感谢字节跳动公司开发了 Trae 这个 AI IDE，让我真的完成了这个愿望。

如果你喜欢这款小工具，欢迎去 GitHub 支持一下，我们后续也会继续更新优化～
